# 🛒 Mercado Mapa

Uma plataforma web prática para localizar mercados e farmácias próximos ao endereço do usuário, facilitando o acesso a itens essenciais.

## 🚀 Funcionalidades
- Busca geolocalizada de estabelecimentos.
- Filtros por categoria (Mercado ou Farmácia).
- Exibição de distância e endereços.


## 🛠️ Tecnologias Utilizadas
- **HTML5/CSS3**: Estrutura e estilização responsiva.
- **JavaScript**: Lógica de busca e integração de dados.
- ## 🛠️ Tecnologias e APIs
- **OpenStreetMap (OSM)**: Utilizado para a renderização dos mapas e dados geográficos.
- **Nominatim API**: Utilizada para Geocoding (converter endereços em coordenadas) e busca de estabelecimentos.
- **Leaflet.js** (biblioteca para exibir o mapa): Biblioteca JS para mapas interativos.
